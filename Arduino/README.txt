iFirmataSerial:
  - You should set the baud rate to your bluetooth device’s baud rate (in the setup method)

iFirmataSoftwareSerial:
  - The Bluetooth device will use pins 2 and 3. You can use the hardware serial for debug purposes.
  - You should set the baud rate to your bluetooth device’s baud rate (in the setup method, iFirmata object).

iFirmataRedBearLabShield
 - The BLEShield from RedBEarLab will use the SPI protocol and will require the Nordic_BLE libraries placed in your Arduino libraries directory. These libraries can be found in: http://redbearlab.com/bleshield/
